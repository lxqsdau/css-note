## <!DOCTYPE html>
> * 全称为Document Type HyperText Mark-up Language
> * 意思为文档种类为超文本标记性语言
> * 此标签可告知浏览器文档使用哪种 HTML 或 XHTML 规范解析
> * 避免出现怪异模式，按统一标准解析html
